# 🛍️ Smart Retail Analytics

**Customer Tracking • Heatmaps • Behavior Analysis**  
Powered by **YOLO11 + ByteTrack**

<!-- ![Demo](results/heatmap_example.jpg) -->

## ✨ Features

- Real-time customer tracking
- Store heatmap generation (popular zones)
- Entry/Exit counting
- Time spent analysis
- Customer flow visualization

## 🛠 Technologies

- YOLO11 (Detection + Tracking)
- ByteTrack Multi-Object Tracker
- OpenCV + Heatmap Generation
- Gradio Web Interface
